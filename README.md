# Clarissa's Portfolio

A modern, responsive portfolio website showcasing projects, experience, and skills. Built with Next.js 16, React 19, and Tailwind CSS.

## Overview

This is a professional portfolio website designed to showcase your work, experience, and technical skills. The site features a clean, modern design with smooth navigation across multiple sections including a home page, project showcase, experience timeline, about section, and contact page.

## Features

- **Responsive Design** - Mobile-first approach with seamless desktop and mobile experience
- **Project Showcase** - Featured projects with detailed descriptions and tech stack information
- **Skills Section** - Display of technical skills and expertise
- **Experience Timeline** - Chronological view of professional experience
- **Dynamic Routing** - Individual project pages with detailed information
- **Modern Tech Stack** - Built with latest versions of Next.js and React
- **Performance Optimized** - Using Next.js for fast page loads and SEO
- **Tailwind CSS Styling** - Utility-first CSS for consistent, beautiful design

## Tech Stack

- **Framework**: [Next.js 16](https://nextjs.org)
- **UI Library**: [React 19](https://react.dev)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com)
- **Language**: TypeScript
- **Linting**: ESLint
